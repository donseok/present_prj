# Upload Template

새 템플릿을 업로드합니다.

## 지원 형식

- **DOCX**: Word 문서 템플릿
- **PPTX**: PowerPoint 프레젠테이션 템플릿

## API 엔드포인트

```
POST /api/templates/upload
Content-Type: multipart/form-data

- file: 템플릿 파일
- name: 템플릿 이름
- documentType: 문서 유형 (제안서, 보고서, 계획서 등)
```

## 플레이스홀더 규칙

템플릿 내 `{{플레이스홀더}}` 형식 사용:
- 기본: `{{프로젝트명}}`, `{{고객사}}`, `{{설명}}`
- 날짜: `{{시작일}}`, `{{종료일}}`
- 목록: `{{팀원목록}}`, `{{마일스톤목록}}`
- 카운트: `{{팀원수}}`, `{{마일스톤수}}`

업로드 시 자동으로 플레이스홀더를 파싱하여 저장합니다.
